🌅 Bing 每日图片 API

提供高质量的 Bing 每日壁纸接口服务

📚 API 接口说明

所有接口返回图片的直接链接，可直接在 img 标签中使用。

GET 今日图片
/api/today

获取今日的 Bing 每日图片

测试今日图片
GET 随机图片
/api/random

从历史图片中随机返回一张

测试随机图片
GET 指定日期图片
/api/date/YYYYMMDD

获取指定日期的图片，例如：/api/date/20231015

🖼️ 图片预览

点击上面的测试按钮查看图片预览

💻 使用示例

HTML 中使用：

<img src="https://你的域名/api/today" alt="Bing今日图片">

JavaScript 中使用：

// 获取今日图片
const todayImage = 'https://你的域名/api/today';

// 获取随机图片
const randomImage = 'https://你的域名/api/random';
